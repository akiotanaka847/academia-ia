Nivel 02 · Conceptos técnicos

JSON y los tipos de datos

Si las APIs son los mensajeros, JSON es el idioma en el que hablan. Es tan legible que lo entenderás de un vistazo, y conocer los tipos de datos te dará el vocabulario para trabajar con cualquier sistema o agente de IA.

El idioma de los datos

01 ¿Qué es JSON?

JSON (JavaScript Object Notation) es una forma de escribir datos que tanto las personas como las máquinas leen con facilidad. Organiza la información en pares de «clave: valor», como una ficha ordenada.

Es el formato en el que casi todas las APIs devuelven su respuesta, y en el que los agentes de IA reciben y entregan información. Reconocerlo es un superpoder cotidiano.

empleado.json
{
  "nombre": "Ana Torres",
  "edad": 34,
  "activo": true,
  "departamento": "Marketing"
}
Cómo leerlo

Clave a la izquierda, valor a la derecha

«nombre» es la clave, «Ana Torres» es el valor. Las llaves { } agrupan una ficha completa. Con eso ya sabes leer el 90% del JSON que verás.

03 Los tipos de datos

Cada valor en JSON es de un tipo. Con estos seis cubres prácticamente todo:

Textostring

  • Va entre comillas
  • "Madrid", "Ana"

Númeronumber

  • Sin comillas
  • 34, 3.14, -5

Booleanoboolean

  • Verdadero o falso
  • true, false

Listaarray

  • Varios valores
  • ["a", "b", "c"]

Objetoobject

  • Ficha anidada
  • { "clave": valor }

Nulonull

  • Ausencia de valor
  • null

Combinados, permiten describir cualquier cosa —incluso datos dentro de datos:

proyecto.json
{
  "proyecto": "Campaña Q3",
  "activo": true,
  "presupuesto": 15000,
  "responsables": ["Ana", "Luis"],
  "cliente": {
    "nombre": "Acme",
    "vip": true
  }
}

04 Por qué te sirve saberlo

  • Depurar automatizaciones: cuando un flujo falla, casi siempre es por leer mal un JSON.
  • Configurar herramientas: n8n, Make y muchas apps usan JSON para sus ajustes.
  • Hablar con la IA: puedes pedirle a un modelo que responda «en formato JSON» para conectarlo a otro sistema.
  • Leer respuestas de API: sabrás exactamente dónde está el dato que necesitas.
Ejercicio · 5 min

Pide un JSON a la IA

Escribe a tu chatbot: «Dame los datos de 3 empleados ficticios (nombre, edad, departamento) en formato JSON.» Observa la estructura de claves y valores que devuelve.

05 Comprueba lo aprendido

1. En "edad": 34, ¿qué es «edad»?

Correcto

«edad» es la clave; 34 es el valor. Clave a la izquierda, valor a la derecha.

2. ¿Cuál de estos es una lista (array)?

Correcto

Los corchetes [ ] indican una lista de varios valores.
