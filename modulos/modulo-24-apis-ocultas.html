Nivel 06 · Web Scraping

Scraping con APIs ocultas

La técnica más limpia de todas. Muchas webs cargan sus datos desde una API interna que devuelve JSON ordenado. Si la descubres, obtienes los datos directamente —sin pelearte con el HTML—. Aquí aprendes a encontrarla.

35 minutos Nivel avanzado Datos limpios en su origen 4 secciones + repaso
La puerta de atrás, ordenada

01 El truco de las APIs ocultas

Cuando abres una web moderna, lo que ves no siempre viene en el HTML: muchas veces la página pide los datos por detrás a una API propia y los pinta al momento. Esa API existe, aunque no esté anunciada. Si la usas directamente, te ahorras leer el HTML: recibes JSON limpio.

Aquí se juntan dos piezas del Nivel 02: las APIs y el JSON. Descubrir la API oculta es, muchas veces, la mejor forma de scrapear.

Por qué es mejor

Más limpio, más rápido, más estable

El HTML cambia de aspecto a menudo y rompe los scrapers. Una API interna suele devolver los datos ya estructurados y cambia menos. Menos esfuerzo, resultados más fiables.

03 Cómo descubrirla, paso a paso

Se hace con las herramientas de desarrollo del navegador (tecla F12), en la pestaña «Network» (Red):

descubrir-api.txt
1. Abre la web y pulsa F12 → pestaña "Network"
2. Filtra por "Fetch/XHR" (las peticiones de datos)
3. Interactúa con la web (busca, cambia de página)
4. Observa qué peticiones aparecen
5. Busca la que devuelve JSON con tus datos
6. Copia su URL → esa es la API oculta

# Ahora puedes pedir esa URL directamente y
# recibir los datos limpios, sin tocar el HTML.

04 Ventajas y cautelas

  • Ventaja: datos estructurados, menos código y mayor estabilidad.
  • Cautela legal: es una API interna, no pública. Aplican las mismas reglas del Módulo 22 (términos, datos personales, volumen).
  • Puede requerir claves: a veces necesita un token o cabeceras; si está protegida, respétalo.
  • Puede cambiar: al no ser pública, el sitio puede modificarla sin avisar.
Cierre del Nivel 06

Poder técnico, criterio responsable

Descubrir una API interna es potente, pero no cambia lo esencial: extrae solo datos legítimos, respeta los términos y no sobrecargues. Con esto cierras el web scraping; en el Nivel 07 aplicamos todo el curso a un caso real con Adobe Workfront.

05 Comprueba lo aprendido

1. ¿Qué es una «API oculta»?

Correcto

Es la API propia del sitio; usarla directamente te da JSON limpio sin parsear HTML.

2. ¿Dónde buscas esas peticiones de datos?

Correcto

La pestaña Network muestra las peticiones; filtrando Fetch/XHR encuentras la que trae el JSON.
